A productivity Android app built with Kotlin and Jetpack Compose, featuring Pomodoro timer and todo list management.

## Features

- Pomodoro Timer
- Todo List
- Settings

## Tech Stack

- Kotlin
- Jetpack Compose
- Android

## Build

```bash
./gradlew assembleDebug
```

## Install

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```
